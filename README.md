## النسخة العربية

**FIRE⚡Calc** هي أداة ويب تفاعلية متقدمة ومصممة بصفحة واحدة (SPA) لمساعدة الأفراد على حساب تاريخ تقاعدهم الدقيق ورقم الثروة المستهدف بناءً على حركات المال العالمية وقاعدة الـ 4% الشهيرة (FIRE Movement).

### 🚀 المميزات (Features)
* **ثنائية اللغة بالكامل:** دعم التحويل الفوري بنقرة واحدة بين العربية والإنجليزية مع تعديل اتجاه الهيكل بالكامل (RTL/LTR).
* **تصميم عصري وسريع:** مبني باستخدام Tailwind CSS لضمان تجربة مستخدم مبهرة ومتجاوبة مع الهواتف والحواسب.
* **رسوم بيانية تفاعلية:** عرض منحنى نمو الثروة التراكمي مقابل الهدف عبر مكتبة Chart.js.
* **آمن بنسبة 100%:** يعمل بالكامل على متصفح المستخدم (Client-Side)، ولا يتم حفظ أو إرسال أي بيانات مالية لأي خوادم خارجية.
* **مجهز لمحركات البحث ومشاركة التواصل:** يحتوي على الكلمات المفتاحية الذكية لـ SEO وأزرار المشاركة الفيروسية على منصة X.

### 🛠️ طريقة التشغيل والرفع (Deployment)
المشروع عبارة عن مِلَفّ واحد مستقل `index.html` ولا يحتاج إلى خادم (Serverless).
1. قم برفع المِلَفّ على استضافات مجانية مثل **Vercel** أو **Netlify**.
2. اربط الدومين الخاص بك مباشرة واستمتع بأداء سريع وصفر تكلفة تشغيل.

## English Version

**FIRE⚡Calc** is a highly professional, modern, single-page interactive web application designed to help users calculate their exact early retirement date and financial freedom target net worth based on the global 4% rule (FIRE Movement).

### 🚀 Key Features
* **Fully Bilingual & Dynamic:** One-click seamless toggle between Arabic and English with full layout direction shifts (RTL/LTR).
* **Modern & Responsive UI:** Crafted with Tailwind CSS for a premium visual experience across all screen sizes (Mobile & Desktop).
* **Interactive Visualization:** Beautiful, dynamic line charts powered by Chart.js showcasing cumulative growth vs. target.
* **100% Private & Secure:** Runs entirely on the client-side. No financial or personal data is collected or sent to external servers.
* **SEO & Viral Ready:** Pre-optimized with meta tags for Google indexing and dynamic sharing buttons for 𝕏 (Twitter).

### 🛠️ Deployment Instructions
Since this is a lightweight, serverless, single-file application (`index.html`), it can be hosted with zero maintenance cost:
1. Connect this repository to **Vercel**, **Netlify**, or **GitHub Pages**.
2. Point your custom domain to the platform, and your automated passive income tool is live!
